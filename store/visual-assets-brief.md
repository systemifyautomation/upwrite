# Store Visual Assets — Brief

All images must be uploaded in the Chrome Developer Dashboard under **Store listing → Images**.

---

## Required

### 1. Screenshots (minimum 1, maximum 5)
- **Dimensions:** 1280×800 px or 640×400 px (16:10 ratio)
- **Format:** PNG or JPEG
- **Do NOT include browser chrome** (no Chrome toolbar visible) — use a clean mockup frame instead

#### Recommended shot list

| # | File name | What to show |
|---|---|---|
| 1 | `screenshot-1-popup-main.png` | The popup open on an Upwork proposal page with the "Generate Proposal" button visible. Show real job data (redact client name if needed). |
| 2 | `screenshot-2-response.png` | The popup after generation — cover letter and question answers rendered in the editable response fields. Use a realistic AI response. |
| 3 | `screenshot-3-recorder.png` | The recorder popup window in `state-recording` state (REC pill + camera PiP bubble visible). |
| 4 | `screenshot-4-autofill.png` | The Upwork proposal form with the cover letter field visibly filled. Split-screen or overlay arrow showing it came from UpWrite. |
| 5 | `screenshot-5-settings.png` | Settings view with the webhook URL field (redact the actual URL — blur or replace with `https://your-n8n.app/webhook/upwrite`). |

**Tip:** Use [Screely](https://screely.com) or Figma to add a device frame around each screenshot.

---

### 2. Small Promotional Tile
- **Dimensions:** 440×280 px
- **Format:** PNG or JPEG
- **Used in:** Chrome Web Store search results grid

#### Suggested design
- Background: solid `#0f0f14` (dark, matches recorder UI)
- UpWrite logo (the `icon128.png` monogram) centered-left, ~80px
- Tagline right of logo: **"AI proposals for Upwork — in seconds"** in white, bold, 22px
- Upwork green (`#14a800`) accent line or badge element

---

### 3. Large Promotional Tile *(optional but strongly recommended)*
- **Dimensions:** 920×680 px
- **Format:** PNG or JPEG
- **Used in:** Featured placement on the Web Store

#### Suggested design
- Same dark background
- Larger logo + tagline
- Mockup of the popup floating over a blurred Upwork proposal page

---

### 4. Marquee Promo Tile *(optional)*
- **Dimensions:** 1400×560 px
- **Used in:** Featured hero slot (rare, editorial choice)

---

## Tools to create these (free options)
- [Canva](https://canva.com) — drag-and-drop, has Chrome extension templates
- [Figma](https://figma.com) — precise, community frames available
- [Screely](https://screely.com) — instant device frame around a screenshot

---

## Notes
- Google does not allow screenshots that contain misleading UI (fake Chrome toolbar, fake browser frame).
- Avoid showing any real user personal data (names, emails, proposal text about real clients).
- Keep text in images readable at thumbnail size — the grid card is small.
